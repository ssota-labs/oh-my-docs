## Scope

Create the Node 24 and pnpm 11.5.2 Turborepo foundation, a minimal Next.js 16
Fumadocs app, canonical product content, templates, planning validation, and CI.

## Implementation sequence

1. Define root workspace and TypeScript contracts.
2. Add the Fumadocs source schema and browser/Markdown routes.
3. Seed the information architecture and linked planning graph.
4. Add graph validation and focused unit tests.
5. Add pull-request validation.

## Verification

- Install with the pinned package manager.
- Run `pnpm check:planning`.
- Run `pnpm test`, `pnpm typecheck`, and `pnpm build`.
- Open `/docs/vision` and `/docs/vision.md`.

## Boundaries

No shared UI package, visual editor, database, authentication, or external
integration is part of this plan.

## Rollback

The foundation has no data migration. Reverting the workspace and docs files
removes the feature.
